howl.io.OutputStream

howl.io.OutputStream
Properties
  • is_closed
    • Methods
  • close
  • write

    • OutputStreams are used for writing to some kind of IO streams. You don’t typically create an output stream yourself, but instead get one from an another source, e.g. Process.stdin.

    Properties

    is_closed

    True if the stream is closed, and false otherwise.

    Methods

    close ()

    Closes the output stream.

    write (data)

    Writes data to the output stream. Raises an error upon any IO error.
